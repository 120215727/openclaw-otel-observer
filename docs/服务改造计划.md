## 🎯 最终架构（已实现 90%）

```
请求入口（快速创建 RawData，status = pending）
    ↓ 立即返回
oc-raw-data
    ↓ 后台线程每 5 秒扫描
RawDataBackgroundProcessor（异步处理）
    ├─→ SessionProcessorService（已实现）
    └─→ Otlp*Processor（TODO）
    ↓
success / failed
    ↓
failed 可通过接口手动重试 → 改回 pending → 后台再次处理
```
